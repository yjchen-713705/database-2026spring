# 数据设计

## teachers
| 场景     | 用途        |
| ------ | --------- |
| 老师教1门课 | 基础        |
| 老师教多门课 | GROUP BY  |
| 老师没有课  | LEFT JOIN |

## courses
| 场景      | 用途        |
| ------- | --------- |
| 课程被多人选  | GROUP BY  |
| 课程只被1人选 | 边界        |
| 课程没人选   | LEFT JOIN |


## students
| 场景     | 用途        |
| ------ | --------- |
| 学生选1门课 | 基础        |
| 学生选多门课 | GROUP BY  |
| 学生没选课  | LEFT JOIN |

## enrollments
| 场景        | 用途     |
| --------- | --------- |
| 高分（90+）   | HAVING |
| 中等（70-89） | 分组     |
| 低分（<60）   | 条件     |
| 同一课程不同成绩 | 平均值    |
